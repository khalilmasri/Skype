## Setup the GUI

The GUI is built using [imgui](https://github.com/ocornut/imgui) and [SDL2](https://www.libsdl.org/) framework.

Ubuntu

        sudo apt-get install libsdl2-2.0 libsdl2-dev

Arch

      sudo pacman -S sdl2


OSX 

        homebrew install sdl2
